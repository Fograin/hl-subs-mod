Bitmap font converter created by: Half-Life Paranoia mod Team

How to use:
1. Don't modify "letters.txt" unless you really now what are you doing.
2. Run "fontconvert.exe" and in new window, choose any font you want (installed on your OS)
3. Choose size and encoding
4. Save file with name "320_Default Text" or "800_Default Text" or "1280_Default Text"
5. Put generated files "X_Default Text.tga" and "X_Default Text.chw" into "\gfx\vgui\fonts\"
