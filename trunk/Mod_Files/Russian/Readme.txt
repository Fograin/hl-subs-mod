HOW TO INSTALL:

1. Unpack 7z archive (using 7Zip,WinRar,WinZip or any other unpacking tool that supports 7z files)
2. Place "sm_valve_ru" directory/folder inside Half-Life 1 directory (e.g. C:\Program Files (x86)\Steam\steamapps\common\Half-Life\)
3. Restart steam
4. Launch "Half-Life: Subtitles Mod 3.0 RU" mod using steam - game library
